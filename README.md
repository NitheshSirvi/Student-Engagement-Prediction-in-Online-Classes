#🎓  Student Engagement prediction in online classes
Overview
A multi-modal educational platform designed to monitor, predict, and improve student engagement in real-time. This project combines Machine Learning (XGBoost) for behavioral prediction and Computer Vision (OpenCV) for live focus tracking. When distraction is detected, the system autonomously deploys "Micro-Interventions" to regain the student's attention.

Built entirely in Python using Streamlit, this enterprise-ready application helps educators process batch data, visualize behavioral trends, and monitor live environments.

#⚠️ Note on Repository Contents
To keep this repository lightweight, secure, and free of machine-specific conflicts, the following folders are intentionally excluded from GitHub (via .gitignore):

venv/: The Python virtual environment containing the heavy installed libraries.

__pycache__/: The compiled bytecode files automatically generated by Python.

When you download this repository, you will need to build your own local virtual environment by following the setup steps below. Once you run the app, Python will automatically generate the __pycache__ files for your specific machine.

#🛠️ Local Installation & Setup Guide
Follow these step-by-step instructions to recreate the environment and run the dashboard on your local machine.

Step 1: Clone the Repository
Download the code to your local machine.

Bash
git clone https://github.com/YourUsername/YourRepositoryName.git
cd YourRepositoryName

Step 2: Create a Virtual Environment (venv)
We need to create a fresh, empty container to hold the project's libraries. Run this command in your terminal:

Bash
python -m venv venv

Step 3: Activate the Virtual Environment
You must activate the environment before installing anything. The command depends on your operating system:

For Windows (Command Prompt or PowerShell):

Bash
venv\Scripts\activate
For Mac / Linux:

Bash
source venv/bin/activate
(You will know it worked when you see (venv) appear at the beginning of your terminal line).

Step 4: Install the Required Libraries
Now that your environment is active, install all the necessary dependencies (like OpenCV, Streamlit, and XGBoost) using the provided requirements file:

Bash
pip install -r requirements.txt
Step 5: Run the Application
Once the installation is complete, you can launch the interactive dashboard!

Bash
streamlit run api/app.py
(Note: Running this command for the first time is what automatically generates the __pycache__ folder on your computer!)

#✨ Key Features
📹 Live Video Proctoring (Vision AI): Uses OpenCV and Haar Cascades to perform real-time facial presence and screen-focus tracking via the user's webcam.

🤖 Predictive Engagement Modeling: Utilizes a trained XGBoost classifier to evaluate tabular behavioral data.

⚡ Automated Micro-Interventions: Closes the feedback loop by automatically triggering simulated pop-up polls when the AI detects a drop in student attention.

📂 Resilient Batch Processing: Allows educators to upload raw CSV files with automated missing-data handling.

📈 Advanced Analytics & Profiling: Generates interactive, enterprise-grade visualizations using Plotly and Statsmodels.

#💻 Tech Stack
